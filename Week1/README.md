# 🤖 Rule-Based AI Chatbot ( Week 1)

A simple and interactive **Rule-Based AI Chatbot** developed using **Python** and **Gradio**. This project demonstrates the basic concepts of Artificial Intelligence by responding to predefined user inputs using conditional (`if-elif-else`) statements. It is designed for beginners to understand chatbot logic without using Machine Learning or Large Language Models.

---

## 📌 Project Overview

This chatbot interacts with users through a clean web interface and provides predefined responses to common questions related to Artificial Intelligence, Python, Machine Learning, and general greetings.

Unlike modern AI assistants, this chatbot is completely **rule-based**, meaning every response is generated through predefined programming logic.

---

## ✨ Features

- 🤖 Interactive chatbot interface using Gradio
- 💬 Responds to predefined user queries
- 👋 Handles greetings
- 🧠 Answers basic AI-related questions
- 🐍 Provides Python information
- 📚 Explains Machine Learning and Deep Learning
- 🚪 Supports exit commands (Bye, Exit, Quit)

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/rule-based-ai-chatbot.git
```

Go to the project folder:

```bash
cd rule-based-ai-chatbot
```

Install the required library:

```bash
pip install gradio
```

Run the notebook in **Google Colab** or **Jupyter Notebook**.

---

## 🚀 How to Use

1. Open the notebook.
2. Run all cells.
3. Launch the Gradio interface.
4. Enter a message in the text box.
5. The chatbot will respond based on predefined rules.

## 🧠 Working Principle

The chatbot follows a simple workflow:

```
User Input
      │
      ▼
Convert to Lowercase
      │
      ▼
Check Using if-elif Rules
      │
      ├── Match Found
      │      │
      │      ▼
      │  Display Response
      │
      └── No Match
             │
             ▼
Display Default Message
```
